Controle Financeiro Pessoa

Sistema de controle financeiro desenvolvido em Java com interface gráfica completa e persistência de dados em banco de dados relacional.

Funcionalidades:
Interface gráfica intuitiva para lançamento de receitas e despesas.
Atualização automática de tabelas.
Cálculo automático de saldo em tempo real.
Inserção de dados em lote utilizando `executeBatch` para maior performance.

Tecnologias Utilizadas:
Java
Oracle Database
JDBC
